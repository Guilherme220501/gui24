Linguagens
Exigidas
Python
Linguagem de programação de alto nível, fácil de aprender e amplamente utilizada em ciência de dados, desenvolvimento web e automação.
JAVA
Linguagem de programação orientada a objetos, conhecida por sua portabilidade entre plataformas e amplamente usada em aplicações empresariais e Android.
Javascript
Linguagem de programação essencial para desenvolvimento web, permitindo a criação de interatividade em páginas e aplicações front-end.
Ruby
Linguagem de programação focada na simplicidade e produtividade, famosa por sua elegância e pelo framework Ruby on Rails para desenvolvimento web.
C
Linguagem de programação de baixo nível, conhecida por sua eficiência e controle sobre recursos de hardware, frequentemente usada em sistemas operacionais e embutidos.
C++
Extensão da linguagem C que suporta programação orientada a objetos, amplamente utilizada em software de sistema, jogos e aplicações de alto desempenho.
C#
Linguagem de programação desenvolvida pela Microsoft, integrada ao .NET Framework, usada para desenvolvimento de aplicações desktop, web e jogos com Unity.
